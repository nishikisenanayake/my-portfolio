# 🌐 Nishiki's Portfolio

A personal portfolio website built to showcase my skills, projects, and experience as a web developer.

---

## ✨ Features

- 👤 **About Me** — A little about who I am and my background
- 🛠️ **Skills** — Technologies and tools I work with
- 💼 **Projects** — A showcase of my work and builds
- 📄 **Resume/CV** — My experience and qualifications
- 📬 **Contact Form** — Get in touch with me directly
